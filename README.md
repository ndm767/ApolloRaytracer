# ApolloRaytracer

A hobby raytracer written in C++ and using SDL2 for output


## Controls

- WASD to move  
- QE to move up and down
- arrow keys for camera rotation
- scenes are loaded using json files, examples can be seen in the assets/scenes folder

## Building

`make [-j#]`  
`./apollo`

## Dependencies

- SDL2
- Assimp
- glm
- nlohmann/json
- cxxopts
